# Welcome to Stakater Cloud Orchestrator Documentation

Welcome to the public documentation for **Stakater Cloud Orchestrator (SCO)**.

Stakater Cloud Orchestrator is a Kubernetes-native control plane designed to help platform teams and service providers design, provision, operate, and govern cloud-native infrastructure and platforms at scale. It enables a consistent, declarative, and secure way to manage Kubernetes environments across their full lifecycle.
